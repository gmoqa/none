Politique de Confidentialité

Dernière mise à jour : 3 novembre 2025

Introduction

Chez NONE, la confidentialité est fondamentale. Cette politique explique clairement quelles données nous collectons, comment nous les utilisons et comment nous les protégeons. NONE est conçu selon le principe de confidentialité dès la conception.

Philosophie de confidentialité

NONE fonctionne entièrement hors ligne par défaut. Toutes vos données (images, audios, configurations) sont stockées uniquement sur votre appareil. Nous n'avons pas de serveurs propres, nous ne collectons pas vos données personnelles et ne les partageons pas avec des tiers.

Données que nous NE collectons PAS

  • Contenu de l'application : Nous n'avons pas accès à vos images, audios ou tableaux créés dans NONE.
  • Informations personnelles : Nous ne collectons pas de noms, e-mails, téléphones ou toute donnée identifiable.
  • Données de santé : Nous ne collectons pas d'informations médicales ou de diagnostic.
  • Localisation : Nous ne suivons pas votre position géographique.
  • Comportement dans l'application : Nous ne surveillons pas comment vous utilisez NONE au sein de l'application.

Données que nous collectons (uniquement sur le site web)

Ce site web (noneapp.org) utilise Google Analytics pour comprendre comment les visiteurs interagissent avec le site :

  • Données de navigation : Pages visitées, temps passé sur le site, pays d'origine (anonymisé).
  • Informations techniques : Type de navigateur, système d'exploitation, type d'appareil.
  • IP anonymisée : Votre adresse IP est anonymisée avant d'être traitée.

Important : Ces données concernent uniquement le site web, PAS l'application mobile.

Google Drive (optionnel)

Si vous choisissez d'activer la synchronisation avec Google Drive dans l'application :

  • Vos données (images, audios, configurations) sont enregistrées dans VOTRE compte Google Drive.
  • NONE n'a accès qu'aux fichiers qu'il crée (dossier "TeaBoard/").
  • Nous (les développeurs) n'avons PAS accès à votre Google Drive.
  • Vous pouvez désactiver cette fonctionnalité à tout moment depuis les paramètres de l'application.

Cookies

Ce site web utilise des cookies uniquement pour Google Analytics :

  • _ga : Identifie les utilisateurs uniques (anonymisés).
  • _gid : Identifie les sessions de navigation.
  • _gat : Limite le taux de requêtes.

Vous pouvez désactiver les cookies depuis les paramètres de votre navigateur.

Utilisation des données

Les données analytiques du site web sont utilisées pour :

  • Comprendre quelles pages sont les plus visitées.
  • Améliorer l'expérience du site web.
  • Savoir depuis quels pays le site est consulté.

JAMAIS nous ne vendons, louons ou partageons des données à des fins publicitaires.

Services tiers

  • Google Analytics : Analyse web (Politique de Google).
  • Google Fonts : Typographie du site web (Politique de Google).
  • YouTube : Vidéos intégrées (Politique de Google).
  • GitHub Pages : Hébergement du site web (Politique de GitHub).

Sécurité

Comme NONE fonctionne hors ligne et que nous n'avons pas de serveurs centraux, vos données sont aussi sûres que votre appareil. Nous vous recommandons de :

  • Utiliser un mot de passe/code PIN sur votre appareil.
  • Maintenir votre système d'exploitation à jour.
  • Si vous utilisez Google Drive, activer l'authentification à deux facteurs.

Confidentialité des mineurs

NONE est conçu pour être utilisé par des personnes de tous âges, y compris les mineurs. Nous ne collectons pas intentionnellement de données de mineurs de moins de 13 ans. Si vous êtes parent/tuteur et pensez que votre enfant a fourni des données personnelles, contactez-nous et nous supprimerons ces informations.

Vos droits (RGPD et réglementations)

Bien que nous ne collections pas de données personnelles depuis l'application, vous avez le droit de :

  • Accès : Demander quelles données analytiques nous détenons (le cas échéant).
  • Rectification : Corriger des données incorrectes.
  • Suppression : Demander la suppression de données.
  • Portabilité : Exporter vos données (vos fichiers sont sur votre appareil/Google Drive).
  • Opposition : Refuser l'utilisation de cookies analytiques.

Modifications de cette politique

Nous pouvons mettre à jour cette politique occasionnellement. Nous vous informerons des changements importants en publiant la nouvelle version sur cette page avec une date mise à jour.

Code source ouvert et transparence

NONE est 100% open source. Vous pouvez consulter le code source complet sur GitHub pour vérifier exactement ce que l'application fait avec vos données.
