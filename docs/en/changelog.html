Changelog

Release history and updates for NONE

v0.1 Initial Release November 2024

Features

  • 12 customizable buttons with pagination (2 pages of 6 buttons)
  • Built-in audio recording (M4A format)
  • Camera and gallery support for button images
  • Edit/Use modes for configuration and daily use
  • Multi-language support (ES, EN, FR, PT, DE, AR, IT, JA, ZH)
  • Local storage with offline-first approach
  • Optional Google Drive sync with automatic fallback
  • Clean, distraction-free interface optimized for accessibility
  • Kotlin Multiplatform architecture (Android + iOS)

Technical

  • Android SDK 34 (min API 21 - Android 5.0+)
  • SwiftUI for iOS implementation
  • Shared constants (ButtonConstants, PreferencesKeys, ValidationConstants)
  • MVP pattern with storage strategy pattern
  • Material Design 3 components
