Política de Privacidade

Última atualização: 3 de novembro de 2025

Introdução

Na NONE, a privacidade é fundamental. Esta política explica claramente quais dados coletamos, como os usamos e como os protegemos. A NONE foi desenvolvida com o princípio de privacidade desde a concepção.

Filosofia de privacidade

A NONE funciona completamente offline por padrão. Todos os seus dados (imagens, áudios, configurações) são armazenados apenas no seu dispositivo. Não temos servidores próprios, não coletamos seus dados pessoais nem os compartilhamos com terceiros.

Dados que NÃO coletamos

  • Conteúdo do app: Não temos acesso às suas imagens, áudios ou pranchas criadas na NONE.
  • Informações pessoais: Não coletamos nomes, e-mails, telefones ou qualquer dado identificável.
  • Dados de saúde: Não coletamos informações médicas ou de diagnóstico.
  • Localização: Não rastreamos sua localização geográfica.
  • Comportamento no app: Não monitoramos como você usa a NONE dentro do aplicativo.

Dados que SIM coletamos (apenas no site)

Este site (noneapp.org) utiliza o Google Analytics para entender como os visitantes interagem com o site:

  • Dados de navegação: Páginas visitadas, tempo no site, país de origem (anonimizado).
  • Informações técnicas: Tipo de navegador, sistema operacional, tipo de dispositivo.
  • IP anonimizado: Seu endereço IP é anonimizado antes de ser processado.

Importante: Esses dados são apenas do site, NÃO do aplicativo móvel.

Google Drive (opcional)

Se você optar por habilitar a sincronização com o Google Drive no app:

  • Seus dados (imagens, áudios, configurações) são salvos na SUA conta do Google Drive.
  • A NONE só tem acesso aos arquivos que ela cria (pasta "TeaBoard/").
  • Nós (os desenvolvedores) NÃO temos acesso ao seu Google Drive.
  • Você pode desativar esta função a qualquer momento nas configurações do app.

Cookies

Este site utiliza cookies exclusivamente para o Google Analytics:

  • _ga: Identifica usuários únicos (anonimizados).
  • _gid: Identifica sessões de navegação.
  • _gat: Limita a taxa de solicitações.

Você pode desativar os cookies nas configurações do seu navegador.

Uso dos dados

Os dados analíticos do site são utilizados para:

  • Entender quais páginas são mais visitadas.
  • Melhorar a experiência do site.
  • Saber de quais países o site é acessado.

NUNCA vendemos, alugamos ou compartilhamos dados para fins publicitários.

Serviços de terceiros

  • Google Analytics: Análise web (Política do Google).
  • Google Fonts: Tipografia do site (Política do Google).
  • YouTube: Vídeos incorporados (Política do Google).
  • GitHub Pages: Hospedagem do site (Política do GitHub).

Segurança

Como a NONE funciona offline e não temos servidores centrais, seus dados estão tão seguros quanto seu dispositivo. Recomendamos:

  • Usar senha/PIN no seu dispositivo.
  • Manter seu sistema operacional atualizado.
  • Se usar o Google Drive, habilitar autenticação de dois fatores.

Privacidade de menores

A NONE foi desenvolvida para ser usada por pessoas de todas as idades, incluindo menores. Não coletamos intencionalmente dados de menores de 13 anos. Se você é pai/responsável e acredita que seu filho forneceu dados pessoais, entre em contato conosco e removeremos essas informações.

Seus direitos (LGPD e regulamentações)

Embora não coletemos dados pessoais do app, você tem direito a:

  • Acesso: Solicitar quais dados analíticos temos (se houver).
  • Retificação: Corrigir dados incorretos.
  • Eliminação: Solicitar a eliminação de dados.
  • Portabilidade: Exportar seus dados (seus arquivos estão no seu dispositivo/Google Drive).
  • Oposição: Recusar o uso de cookies analíticos.

Alterações a esta política

Podemos atualizar esta política ocasionalmente. Notificaremos alterações importantes publicando a nova versão nesta página com uma data atualizada.

Código aberto e transparência

A NONE é 100% código aberto. Você pode revisar o código-fonte completo no GitHub para verificar exatamente o que o app faz com seus dados.
